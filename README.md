# Eruru.JsonConfig

A high-performance, thread-safe JSON configuration system with asynchronous read/write support and automatic hot reload.
Built on top of `System.Text.Json` and `JsonTypeInfo` (Source Generator), with native AOT (Ahead-of-Time Compilation) support.  
Suitable for .NET backend services, desktop applications, game servers, plugin systems, toolchains, and other scenarios requiring dynamic configuration management.

一个高性能、线程安全、支持异步读写与自动热重载的 JSON 配置系统。  
基于 `System.Text.Json` 与 `JsonTypeInfo`（Source Generator）构建，原生支持 AOT（Ahead-of-Time Compilation），适用于 .NET 服务端、桌面应用、游戏服务器、插件系统、工具链等需要动态配置管理的场景。
